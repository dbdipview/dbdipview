This folder is a template for processing project upon acceptance of records.
Copy the content into your work folder. Example naming convention: 000_NAME (for ingest #000 with submitter NAME)
The folders:

data:
	Delete if there will be no files packed into EDDV package.

metadata: 
	Use the provided templates as a starting point for your metadata.
	Consider using automated tool to generate metadata, if applicable.

package:
	Store any "external" files (not part of the DDV/EDDV package) in this folder during the test phase.
	In addition, there is also an empty order file. Suggested name syntax: 000_NAME_order.xml 

processing_docs:
	Document each processing step and workflow iteration in this folder.

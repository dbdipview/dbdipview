Folder for external files to be referenced by the queries.
The name of the folder is the name of dbDIPview package file.
It is independent from the database name.
In queries.xml, the file names are in elements <ahrefs> and <images> where 
the exact column name with file name is in <dbcolumnname>.

Example:

dbdipview/www/files/container__viewer/picture1.png
